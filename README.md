# laravel-photo-upload

Application to upload files with input and with api, to display uploaded files with date

Installation:

1. Clone GitHub repo for this project
2. cd into your project
3. Install Composer Dependencies >composer install
4. Create a copy of .env file >cp .env.example .env
5. Generate an app encryption key >php artisan key:generate
6. Create the symbolic link >php artisan storage:link
7. Start application >php artisan serve

Application will be accessed using /public directory
